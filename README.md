# Webui-template
基于gradio界面的webui.py模板。如果不想自己做界面可以套用模板：很简单，把后端程序和webui模板塞给ai让它帮你搓一个webui.py就可以了

优势： 1.使用webui降低使用门槛  

       2.套用模板降低前端的编写成本  
      
       3.避免调试时遇到各种奇奇怪怪bug，降低纠错的烦恼

劣势：1.不够精美，不够针对，只能用在一些泛用性比较高的场景
